# /orchestrate - Multi-Agent Coordination

$ARGUMENTS

---

## Behavior

1. **Discovery & Planning**: Activate `project-planner` to create blueprint.
2. **Specialist Selection**: Delegate sub-tasks to domain agents (`backend-specialist`, `frontend-specialist`, `database-architect`, etc.).
3. **Execution & Quality Gate**: Verify all code changes using `quality-gatekeeper` and automated tests.
