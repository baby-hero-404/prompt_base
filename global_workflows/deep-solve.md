## Behavior

When `/deep-solve` is triggered:

1. Analyze intent across multiple domains.
2. Load matching skills dynamically from `registry.min.json`.
3. Synthesize architectural blueprint.
4. Execute sub-tasks with specialized agents.
5. Prune loaded skills from context window once task completes.
